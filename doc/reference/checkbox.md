# Checkbox

Toggleable boolean input with label.

**Constructor:** `NewCheckbox(id, class, text string, checked bool) *Checkbox`

## Methods

- `Toggle()` — switches checked state

## Events

| Event | Data | Description |
|-------|------|-------------|
| `"change"` | `bool` | Checked state changed |

## Notes

Flags: `"checked"`, `"focusable"`, `"readonly"`
